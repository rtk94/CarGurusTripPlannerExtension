# CarGurus Trip Planner Extension

A standalone browser extension to manage and visualize your saved CarGurus listings.

## Features
- **One-Click Sync:** Adds a button to your CarGurus Saved Cars page.
- **Standalone Dashboard:** Built-in map and sidebar dashboard.
- **Local Storage:** No server required. All data is saved in your browser.
- **CSV Export:** Download your car data anytime.

## Installation (Unpacked)
1. Open Chrome and navigate to `chrome://extensions/`.
2. Enable **Developer mode** (top-right toggle).
3. Click **Load unpacked**.
4. Select the `[Project Folder]` folder.

## Usage
1. Go to your [CarGurus Saved Cars](https://www.cargurus.com/Cars/saved-listings) page.
2. Click **🚀 Sync to Car Map**.
3. The dashboard will open automatically in a new tab.
